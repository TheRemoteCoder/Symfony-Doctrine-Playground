## Setup

### Homestead

#### MacOS

- MacOS: Update native PHP version 7.3x to 7.4 via Homebrew
  - Needs a lot of custom configuration + Re-Aliasing the `php` command

#### Use

Useful commands after successful installation of Vagrant + Homestead.

```txt
# Start + login
vagrant up
vagrant ssh

# Default access
mysql -u homestead -p
# secret
```
